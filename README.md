Symfony React Boilerplate
--

#### Clone this repository using

```bash
git clone https://github.com/rummykhan/symfony-react.git
```

#### Install dependencies

1. `composer install`
2. `npm install`
3. `php bin/console server:run`
4. `npm run watch` or `npm run production`
